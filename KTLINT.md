# Ktlint Integration

Этот проект настроен с автоматическим форматированием кода с помощью ktlint.

## Команды

- `./gradlew ktlintFormat` - Форматирует код согласно правилам ktlint
- `./gradlew ktlintCheck` - Проверяет код на соответствие правилам ktlint
- `./gradlew check` - Запускает все проверки, включая ktlint

## Автоматическое форматирование

ktlint автоматически запускается при выполнении задач:
- `build`
- `test`
- `check`

## Конфигурация

- Версия ktlint: `12.3.0`
- Android стиль: включен
- Игнорирование ошибок: включено (для плавной интеграции)
- Исключены папки: `**/generated/**`, `**/build/**`

## Файлы конфигурации

- `.editorconfig` - Настройки форматирования для IDE
- `build.gradle.kts` - Основная конфигурация ktlint
- `composeApp/build.gradle.kts` - Конфигурация для модуля composeApp
- `gradle/libs.versions.toml` - Версии плагинов

## Исправление ошибок

Большинство ошибок форматирования исправляются автоматически при запуске `./gradlew ktlintFormat`. 
Некоторые ошибки требуют ручного исправления (например, имена функций, которые должны начинаться с заглавной буквы для Compose функций).
